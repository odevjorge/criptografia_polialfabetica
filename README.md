## Algoritmo de Criptografia Polialfabetica

É um algoritmo onde você define uma quantidade de saltos e uma quantidade de ciclos, onde:

Saltos: É quantidade de casas que uma letra vai pular ex:
- salto = 3
- A -> D

Cilos: É a quntidade de vezes que ele vai trocar as letras ex: pegando o slato de cima
- Ciclo = 2
- A -> D -> G

O prograva vai pegar uma quantidade de saltos e ciclos que você vai definir ou vai ser gerado aleatoriamente.

Em seguida você vai mandar a OPÇÃO(Criptografar ou Descriptografar) e vai mandar uma PALAVRA ou fraze para ser feito o processo.
